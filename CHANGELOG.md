## [v0.1.0] - 2026-03-21

### Highlights

- CLI tool to convert TypeScript Zod schemas to Go structs via JSON Schema intermediate format
- Full pipeline with `generate` command, or step-by-step with `zod2json` and `json2go` commands
- Complete Insomnia v5 schema conversion example included

### Added

- JSON Schema parser with support for objects, arrays, enums, refs, nullable types ([`42b1180`](https://github.com/grokify/zod2go/commit/42b1180))
- Go code generator with json/yaml tags, pointers for optional fields, and enum constants ([`42b1180`](https://github.com/grokify/zod2go/commit/42b1180))
- Zod to JSON Schema conversion via Node.js integration (tsx, ts-node, or node) ([`42b1180`](https://github.com/grokify/zod2go/commit/42b1180))
- `generate` CLI command for full pipeline: Zod → JSON Schema → Go ([`67b9bd7`](https://github.com/grokify/zod2go/commit/67b9bd7))
- `zod2json` CLI command to convert Zod schema to JSON Schema ([`67b9bd7`](https://github.com/grokify/zod2go/commit/67b9bd7))
